# svelte-standalone-starter

Full **Svelte 3** **TypeScript** starter with automatic bundling into a minified **single HTML file** featuring:

- Svelte
- Vite
- TypeScript
- TailwindCSS
- SCSS
- Autoprefixer
- HTML, SCSS and JS minification
- SCSS and JS inlining to HTML
- ESLint
- Prettier
- Stylelint
- Yarn

## develop

**`yarn dev`**

## lint

**`yarn lint`**

## format

**`yarn format`**

## build

**`yarn build`**
